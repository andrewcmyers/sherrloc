Diagnosis of Type Inference Failures

This web page demonstrates a general method for diagnosing errors found in computer programs. In this case the method is being applied to OCaml programs.

Limitations

  • Programs spanning multiple files are not supported.
  • Modules are not fully supported.
  • The program must be self-contained, though it can use some standard libraries.
  • Array constants and hexadecimal constants are not supported.

How to use it

  1. Read the notice below about how we will use your code.
  2. Paste your code into the box on the left and press the “Check it!” button.
  3. A list of detected type errors will be provided. By moving the mouse over the errors, you can see what part of the program is determined to be the likely cause of the error.
