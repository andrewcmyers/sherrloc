Module Print (.ml)


module Print: sig .. end
This module provides a simple pretty-printer for the terms maintained by a unifier.
val reset : unit -> unit
reset() clears print's memoization table.
val print_variable : ?user_name_from_int:(int -> string) ->
       bool -> MultiEquation.variable -> string
print context x returns a printable representation of the object x. Consecutive calls to print share the same variable naming conventions, unless reset is called in between. The context context may be used to pass additional information to the printing function.
val print_term : ?user_name_from_int:(int -> string) -> bool -> MultiEquation.crterm -> string
